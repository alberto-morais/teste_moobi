## Usage
Pode configurar direto em .env tem o .env.example para seguir como exemplo
```php
DB_DRIVER=mysql
DB_HOST=localhost
DB_DATABASE=moobi_toy
DB_USERNAME=root
DB_PASSWORD=123456

PASS=mudar123456

BASE_URL=http://localhost/moobitoy/
DOMAIN=localhost/moobitoy/
```

```php

senha: 123456
```
